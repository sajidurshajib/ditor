# DITOR 
### Like a simple editor

## Before you use 
```
sudo ditor /usr/bin -r
```
copy this into bin folder then you can run this script anywhere. 

## Command
```
ditor filename.txt command
```
```
-r = read file
-rl = read file with number of lines
-c = copy all text
-cl=10 = copy 10th line
```
